# kakaoDreamy
## 💬 프로젝트 설명

제주대학교 학생들이 자신의 졸업 학점을 계산하는데 어려움을 겪고 있어서 도움을 주고자 웹 사이트를 개발하게 되었습니다. 

## ⚙️ 기능

- 학점 계산
- 현재 학기 시간표
- 수강 편람
